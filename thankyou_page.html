Learn more about Code Scouts

Code Scouts is an Oregon Nonprofit startup founded in 2012 with the mission to empower women to become software developers. Code Scouts builds local communities of aspiring developers who create customized learning plans with mentors, using existing learning resources to hone their skills alongside other learners.

Learn more about Stumptown Syndicate

The mission of Stumptown Syndicate is to create resilient, radically inclusive tech and maker communities that empower positive change. All Stumptown Syndicate programs are volunteer-run, community-oriented, peer-to-peer learning events. Stumptown Syndicate hosts events whose intended audience ranges from those with no technical knowledge, to those who work with technology professionally.
